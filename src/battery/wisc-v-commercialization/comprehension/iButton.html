Administration

Start
Ages 6–11: Item 1
Ages 12–16: Item 3
Children suspected of having an intellectual disability or low cognitive ability should start with Item 1.

Reverse
If a child aged 12–16 does not obtain a perfect score on either of the first two items given, administer the preceding items in reverse order until the child obtains perfect scores on two consecutive items.

Discontinue
After 3 consecutive scores of 0

Recording

Don’t Know (DK): Used to record when the child provides an answer of “I don’t know” as his or her only response

No Response (NR): Used to record when the child does not respond

Query Administered (Q): Used to record when any query is provided

Item Repetition (IR): Used to record when an item is repeated

†Teaching: Used to access corrective feedback for imperfect scores on Items 1, 2, 3, & 4

§< 2 General Concepts: Used to access queries for items with multiple general concepts

Scoring

Score 0, 1, or 2 points.

Refer to the WISC–V Administration and Scoring Manual for additional information.
