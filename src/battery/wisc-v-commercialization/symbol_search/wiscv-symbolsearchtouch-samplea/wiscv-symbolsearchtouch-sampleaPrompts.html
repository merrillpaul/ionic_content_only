Say, Now you do these (point to sample items). If you see the shape, touch it. If you don’t see the shape, touch the NO box. Then you can work on the next one.

Say, Go ahead. Allow the child to complete the sample items. If the child responds correctly, the test will advance to the next sample item. The test will not advance if the sample item is answered incorrectly. If the child responds incorrectly, provide feedback as described below. 

Correct: 

Say, That’s right. Now you know how to do them. Proceed to Test Items.

Incorrect:

Correct the error(s) immediately, as follows:

If the child incorrectly touches NO on the first or third sample item, say, That’s not quite right. This shape (point to target symbol) is the same as this one (point to matching symbol in search group), so you should touch it. Do NOT touch the response during teaching. Allow the child to touch the response. Teach until the child responds correctly.

If the child incorrectly touches a non-target symbol in the search group on the first or third sample item, say, That’s not quite right. This shape (point to error) is not the same as this one (point to target symbol), so you should not touch it. Do NOT touch the response during teaching. Allow the child to touch the response. Teach until the child responds correctly.

If the child incorrectly touches the rotated target symbol on the second sample item, say, That’s not quite right. Even though this shape (point to rotated target in search group) looks like this one (point to target symbol), you should not touch it because it’s turned the wrong way. You should touch the NO box. Do NOT touch the response during teaching. Allow the child to touch the response. Teach until the child responds correctly.

If the child incorrectly touches any other symbol in the search group on the second sample item, say, That’s not quite right. This shape (point to error) is not the same as this one (point to target symbol), so you should not touch it. Do NOT touch the response during teaching. Allow the child to touch the response. Teach until the child responds correctly.

If it is clear that the child will not correctly complete the sample items or understand the task with further instruction, discontinue the subtest. 

When the child has successfully completed the sample items, proceed to Test Items.
