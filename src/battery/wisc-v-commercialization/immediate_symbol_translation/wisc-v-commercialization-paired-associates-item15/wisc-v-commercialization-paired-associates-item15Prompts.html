Say, Now look at these. Point to the and say, This one means YESTERDAY, and this one (point to ) means ABOUT.

This one (point to ) is special. If you see it on top of another symbol, it makes it PAST TENSE, like it already happened. So if you see this one (point to  in gray box) on top of the one for SEE (point to in gray box), you say SAW.

This one (point to ) means BOOK, and this one (point to ) means HOT. Now you say them after me.

Point to each symbol as you translate it aloud and pause to allow the child to respond, YESTERDAY…ABOUT…make PAST TENSE…BOOK…HOT. Immediately correct any translation errors. If necessary, encourage the child to respond.
