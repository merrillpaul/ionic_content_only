Open the Response Booklet to page 3 and place it in front of the child.

Point across the shapes in the key at the top of the page and say, Look at these shapes. Point to the mark inside of each shape as you say, Each shape has its own special mark.

Point to the demonstration items and say, Down here, the shapes are empty. I will show you how to draw the right mark in each shape.

Point to the first demonstration item (circle) and say, Here is a circle. The circle has this mark (point to corresponding symbol in key), so I draw it like this (draw symbol).

Point to the second demonstration item (star) and say, Here is a star. This star has this mark (point to corresponding symbol in key), so I draw it like this (draw symbol).
