Client Device
3 x 4 Response Grid
6 Response Chips

Administration

Start
Ages 5-8: Delayed Recall

Discontinue
Do not discontinue

Recording

Don't Know (DK): Used to record when the examinee provides an answer of "I don't know" as his or her only answer

No Response (NR): Used to record when the examinee does not respond

Prompt (P): Used to record when any subtest-specific prompt is provided

Repetition (R):Used to record when the examinee indicates a need for repetition

Scoring

Delayed Recall: 0-6 points

Refer to the CMS Manual for additional information.
