Materials

Ages 5-8
       Client Device (Raised)
       3 x 4 Response Grid
       6 Response Chips

Ages 9-16
       Client Device (Raised)
       4 x 4 Response Grid
       8 Response Chips

Administration

Start
Ages 5-8: Trial 1 (Card A)
Ages 9-16: Trial 1 (Card C)

Discontinue
Do not discontinue.

Timing

Present the stimulus for Trials 1-4 for 5 seconds using the on-screen timer. 

Recording

Don't Know (DK): Used to record when the child provides an answer of "I don't know" as his or her only answer

No Response (NR): Used to record when the child does not respond

Prompt (P): Used to record when any subtest-specific prompt is provided

Repetition (R): Used to record when the child indicates a need for repetition

Scoring

Responses are scored by entering the chips from the examiner's point of view.

Trials 1-3 and Immediate Recall
Ages 5-8: 0-6 points
Ages 9-16: 0-8 points

Note: Trial 4 is not scored. 

Refer to the CMS Manual for additional information.
