Materials

Client Device (Flat)
Response Booklet
Pencils with erasers

Administration

Start
Delayed Recall Item 1

Discontinue
Do not discontinue.

Timing

Administer 10–30 minutes after Visual Reproduction I.

Use the provided timer to record how long it takes the examinee to reproduce the design.

Recording

The examinee records his or her responses in the Response Booklet.

Don’t Know (DK): Used to record when the examinee provides an answer of “I don’t know” as his or her only answer

No Response (NR): Used to record when the examinee does not respond

Prompt (P): Used to record when a subtest-specific prompt is provided

Repetition (R): Used to record when the examinee indicates a need for repetition

Scoring

For Delayed Recall and Copy, select “Attempted” if the examinee draws anything, regardless of whether it is correct. Select “No drawing” if the examinee does not draw anything for the item.

For Delayed Recognition, select the examinee’s response as “Yes” or “No.”

Refer to the WMS 5 Digital Pilot Manual for additional information.
