Administration

Start
Ages 65-90: Story A

Discontinue
Do not discontinue.

Recording

Don’t Know (DK): Used to record when the examinee provides an answer of “I don’t know” as his or her only answer

No Response (NR): Used to record when the examinee does not respond

Prompt (P): Used to record when a subtest-specific prompt is provided

Repetition (R): Used to record when the examinee indicates a need for repetition

Query (Q): Used to record when any query is provided

Scoring

Score 0 or 1 point for each story detail.

Note that the scoring procedures for this subtest may require the examiner’s judgment. Examples of 1-point and 0-point responses are listed in the Samples pop-over next to the 1-point criteria. 

Refer to the WMS-IV Administration and Scoring Manual for additional information.
