Transition to the TFLS app on the Client Device, and select Microwave at the start page. Place Card 2. Food Label in front of the examinee. Say, On this card (point to the Food Label) is a label that tells you how to cook macaroni and cheese in the microwave (point to the microwave). Use the directions on the label to show me how to set the microwave so the food cooks correctly.

The macaroni and cheese appears in the oven window when the Start button is touched.

When the item is complete, select Exit to return to the start page.
