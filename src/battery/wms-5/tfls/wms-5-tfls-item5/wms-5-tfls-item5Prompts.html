Transition to the TFLS app on the Client Device, and select Online Shopping at the start page. Place the TFLS Express Card face up in front of the examinee. Say, Pretend this is your credit card, and you are going to shop at this online store (point to online store). Show me how you would select a shirt, use the credit card to pay for the shirt from this store, and have the shirt shipped to your home.

When the item is complete, select Save to save a copy of the responses or Exit to return to the start page. 

Prompts

  • If the examinee attempts to select a category other than Clothing or appears to be browsing the site, say, You are only buying a shirt.
  • If the examinee is unable to navigate to the Shirt Selection page, navigate to the page and say, Now you select a shirt. Select Shirt Page to indicate the prompt was given.
  • If the examinee is unable to select a shirt from the Shirt Selection page, choose any shirt and say, Now you complete the purchase. Select Shirt Select to indicate the prompt was given.
  • If the examinee is unable to proceed to the Checkout page after selecting a shirt, navigate to the page and say, Now you finish.  Select Checkout to indicate the prompt was given.
  • If the examinee asks if they should use their name, address, and phone number for the shipping information, say, Yes.
  • The examinee has two opportunities to enter the correct credit card number. On the second attempt, the review pop-up will appear regardless of the accuracy of the credit card number.
  • If the examinee is unable to select Place your order after proceeding to Checkout page, complete the order for them. Select Place Order to indicate the prompt was given.
