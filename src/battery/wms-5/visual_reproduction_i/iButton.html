Materials

Client Device (Flat)
Response Booklet
Pencils with erasers

Administration

Start
Item 1

Discontinue
Do not discontinue.

Timing

Present each item stimulus for 10 seconds.

Use the provided timer to record how long it takes the examinee to reproduce the design.

Administer Visual Reproduction II 10–30 minutes after completing Visual Reproduction I.

Recording

On Item 1, Trial 1, record which hand the examinee used to draw the design.

Don’t Know (DK): Used to record when the examinee provides an answer of “I don’t know” as his or her only answer

No Response (NR): Used to record when the examinee does not respond

Prompt (P): Used to record when a subtest-specific prompt is provided

Repetition (R): Used to record when the examinee indicates a need for repetition

Scoring

The examinee draws in the Response Booklet.

For each trial and item, select “Attempted” if the examinee draws anything, regardless of whether it is correct. Select “No drawing” if the examinee does not draw anything for the trial.

Refer to the WMS 5 Digital Pilot Manual for additional information.
