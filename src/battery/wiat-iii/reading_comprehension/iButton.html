Reverse
Grades 2-12+: If the sum of scores for all items corresponding to the first story after a start point is 2 or less, go back one start point and continue administration. The reverse rule may be applied a maximum of 3 times; do not go back more than 3 start points from the grade-appropriate start point.

Recording

Don't Know (DK): Used to record when the examinee provides an answer of "I don't know" as his or her only answer

No Response (NR): Used to record when the examinee does not respond

Prompt (P): Used to record when a subtest-specific prompt is provided

Repetition (R): Used to record when the examinee indicates a need for repetition

Query (Q): Used to record when any query is provided. 

Scoring

1 or 2 points are awarded for each correct response and 0 points awarded for each incorrect, DK, or NR response.

Refer to the WIAT-III Examiner's Manual for additional information.
