Administration

Start
Grades 1-12+: Sample A

Discontinue
If score is 0 on first 2 items, record a raw score of 0 for Sentence Building.

Recording

Don't Know (DK): Used to record when the examinee provides an answer of "I don't know" as his or her only answer

No Response (NR): Used to record when the examinee does not respond

Prompt (P): Used to record when any subtest-specific prompt is provided 

Repetition (R): Used to record when the examinee requests that the item be repeated

Scoring

Detailed scoring procedures are provided on screen for each item and can be accessed from All To Do post-administration. Scoring information regarding basic rules for written grammar and mechanics is located in Appendix B.2 of the WIAT-III Examiner's Manual.

Refer to the WIAT-III Examiner's Manual for additional information.
