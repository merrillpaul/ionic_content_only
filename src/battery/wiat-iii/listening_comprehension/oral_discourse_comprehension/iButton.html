Materials

Client Device (Raised)

Administration

Start
Grades PK-3: Item 1
Grades 4-12+: Item 6

Reverse
Grades 4-12+: If score of 0 on any of the first 3 items given, administer preceding items in reverse order until 3 consecutive scores of 1. 

Discontinue
After 4 consecutive scores of 0

Recording

Don't Know (DK): Used to record when the examinee provides an answer of "I don't know" as his or her only answer

No Response (NR): Used to record when the examinee does not respond

Prompt (P): Used to record when any subtest-specific prompt is provided 

Repetition (R): Used to record when the examinee requests that the item be repeated

Query (Q): Used to record when any query is provided. Verbatim instructions for queries can be found by selecting the picklist buttons marked with a Q. 

Scoring

1 point is awarded for each correct response and 0 points awarded for each incorrect, DK, or NR response. 

Refer to the WIAT-III Examiner's Manual for additional information.
