Administration

This subtest contains abbreviated instructions followed by full instructions. Use the abbreviated instructions if you are administering Math Fluency - Multiplication immediately following another Math Fluency subtest. Use the full instructions if there is a delay between the subtests or if another Math Fluency subtest has not been administered.

Note. The examinee may not use a calculator.

Timing

Begin timing after the last word of instructions. Stop timing when the examinee finishes all of the problems or after 60 seconds.

Recording

Don't Know (DK): Used to record when the examinee provides an answer of "I don't know" as his or her only answer

No Response (NR): Used to record when the examinee does not respond

Prompt (P): Used to record when a subtest-specific prompt is provided

Repetition (R): Used to record when the examinee indicates a need for repetition

Scoring

Use the answer keys provided in Appendix C to score each response 1or 0 points.

When the examinee’s response matches the correct response listed in the answer key, record a small check mark beside the item in the Response Booklet. When the examinee’s response does not match the correct response listed, record a small X beside the item in the Response Booklet.

Refer to the WIAT-III Examiner's Manual for additional information.
