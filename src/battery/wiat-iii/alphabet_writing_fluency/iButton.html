Materials

Response Booklet
Pencil without eraser

Administration

The examinee may write letters however he or she is most comfortable: in upper- or lowercase, in print or cursive. Use the interface to capture whether each letter is written in upper- or lowercase font. 

Start
Grades PK-3: Item 1

Timing

Allow a maximum of 30 seconds to complete. If all letters are written prior to the time limit, or the examinee cannot write any more letters, stop the timer.

Recording

Don't Know (DK): Used to record when the examinee provides an answer of "I don't know" as his or her only answer

No Response (NR): Used to record when the examinee does not respond

Prompt (P): Used to record when a subtest-specific prompt is provided

Repetition (R): Used to record when the examinee indicates a need for repetition

Scoring

1 point for each correct letter as defined in appendix B.1 of the WIAT-III Examiner's Manual.

Refer to the WIAT-III Examiner's Manual for additional information.
